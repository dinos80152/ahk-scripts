# AutoHotKey Script Examples

## SublimeText

* Packages -> AutoHotKey
  * Ctrl + b
  * F1
  * Ctrl + Alt + H

* **For Traditional Chinese Windows, File has to save in BIG5 encoding**

## Notice

* Always put this code in script for force quit

```
Escape::
ExitApp
Return
```
